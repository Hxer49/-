# Portability And Backup

This learning archive is designed to survive computer changes.

## What To Copy

Copy the entire folder:

`digital-power-learning`

This folder contains the source of truth:

- `learning-profile.md`
- `roadmap-24-months.md`
- `progress-tracker.md`
- `daily-log.md`
- `tomorrow-plan.md`
- `resources.md`
- `AGENTS.md`
- `PORTABILITY.md`

The personal skill is optional but useful:

`C:\Users\rcrt\.codex\skills\digital-power-coach`

If moving to another computer, copy that skill folder to the new Codex skills directory if possible.

## Recommended Backup Method

Best option:

- Put `digital-power-learning` in a Git repository and push it to a private GitHub repo.

Simple option:

- Zip the `digital-power-learning` folder after each week of study.
- Store it in cloud drive or an external disk.

Manual option:

- Copy the folder to the new computer.
- Open Codex from inside the copied folder.
- Ask: "Read this archive and continue my digital power learning plan."

## After Moving To A New Computer

1. Open Codex in the copied `digital-power-learning` directory.
2. Make sure `AGENTS.md` is present.
3. Ask Codex to read:
   - `learning-profile.md`
   - `progress-tracker.md`
   - `daily-log.md`
   - `tomorrow-plan.md`
4. If the `digital-power-coach` skill is unavailable, the archive still works because the files contain the core instructions.

## Important Rule

Do not treat any absolute path as permanent. If this folder moves, the markdown files in the current directory are the source of truth.
