# Resource Notes

This file stores useful resources to review later. Add links, books, application notes, videos, and tool notes here as the learning path develops.

## Tool Candidates

- PSIM or PLECS for power electronics simulation.
- MATLAB/Simulink for control-system understanding.
- Code Composer Studio for TI C2000.
- STM32CubeIDE or current STM32 workflow for comparison projects.

## Hardware Candidates

- TI C2000 LaunchPad: F280049C, F280039C, or F28379D.
- Low-voltage Buck converter module or development board for safe bring-up.

## Notes

- Prefer low-voltage, current-limited experiments first.
- Do not connect high-power hardware without a safety checklist and proper instruments.
